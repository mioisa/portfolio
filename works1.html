    4ヶ月

    Illustrator / Photoshop / XD / HTML / CSS / JavaScript / jquery

メインページ

①メインカラーはHeartilyのイメージカラーのピンクを使用。
清潔感、文字や写真の見やすさを意識して、薄めのピンクを選びました。

②PRODUCT（取扱商品）や、HEARTILY（サロンケア）を部分別にカテゴリー化。
お客様の目的に合った商品をすぐに探せるようにしました。

③使用する写真は、色合いや雰囲気に統一感を持たせるようシンプルなものにしました。

④ボタンや写真に合わせて、アイコンも立体感があるものを選びました。

⑤メインページは、一目見て事業内容が分かるようにシンプルな内容にしました。
